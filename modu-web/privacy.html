隐私协议

最后更新：2026 年 6 月 30 日

1. 我们收集的信息

Modu 网站当前主要用于展示项目介绍、安装方式和开源仓库入口。我们不会在本站主动要求你注册账号，也不会通过表单收集个人身份信息。

2. 自动产生的访问数据

当你访问本站时，托管服务、内容分发网络、浏览器或安全服务可能会自动处理必要的技术信息，例如 IP 地址、访问时间、浏览器类型、设备信息、请求路径和错误日志。这些信息用于保障网站加载、安全防护和基础运维。

3. Cookie 和本地存储

本站源码没有内置用于广告追踪的 Cookie 逻辑。若第三方托管、字体、统计或安全服务设置 Cookie，其处理规则以对应第三方的政策为准。

4. 第三方链接

本站包含指向 GitHub 以及其他第三方网站的链接。你访问第三方网站时，其隐私政策和服务条款将独立适用，请在使用前自行查看。

5. 第三方共享与披露

为运行本站和维护开源项目，我们可能在必要范围内与以下第三方处理或共享技术数据：

  • Cloudflare Pages、CDN 与安全服务：用于网站托管、访问加速、安全防护、日志记录和故障排查，可能处理 IP 地址、请求路径、浏览器信息、访问时间和安全事件数据。
  • GitHub：用于代码托管、Issue 支持、项目协作和发布管理。你在 GitHub 上主动提交的用户名、头像、issue、评论和联系方式由 GitHub 按其规则处理。
  • Google Fonts：用于加载网页字体，浏览器可能向 Google 发送字体资源请求及必要的技术信息。
  • 域名解析、DNS 与基础设施服务商：用于解析域名、保障网站可访问性和排查网络问题，可能处理必要的 DNS 查询和访问元数据。

除上述运行、支持、安全和法律合规目的外，我们不会出售你的个人信息，也不会将访问数据用于广告画像。

6. 信息使用

我们仅在必要范围内使用访问数据，用于网站运行、安全排查、性能分析、问题定位和项目维护。如果你通过 GitHub 或其他渠道主动联系我们，我们会使用你提供的信息处理对应请求。

7. 用户数据权利

在适用法律允许的范围内，你可以请求查阅、更正、删除或限制处理与你相关的个人数据，也可以就我们的数据处理方式提出疑问或投诉。由于本站当前不提供账号系统，直接由我们保存的个人数据通常仅限你主动提交的支持请求信息；若数据由 GitHub、Cloudflare、Google 等第三方服务处理，你可能需要直接向对应服务商行使相关权利。

你可以通过 GitHub Issues 在线支持 提交数据权利请求。提交请求时请说明你希望查阅、更正或删除的数据范围，并避免公开提交敏感信息。

8. 数据安全

我们会采用合理措施保护与本站相关的数据安全。但互联网传输和第三方平台运行无法保证绝对安全，请避免通过公开渠道提交敏感信息。

9. 协议更新

我们可能会根据网站功能、托管方式或项目维护需要更新本协议。更新后的版本会发布在本页面，并以页面显示的更新日期为准。

10. 联系方式

如果你对隐私事项、数据权利或支持请求有疑问，可以通过 GitHub Issues 在线支持 与项目维护者联系。请不要在公开 issue 中提交密码、API Key、访问令牌或其他敏感信息。
